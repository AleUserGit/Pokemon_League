# PokéTrainer League
## Descripción del programa
Sistema de gestión y almacenaje de Pokemones. El programa recibe los datos de Nombre, Tipo, Nivel, Poder, Entrenador, Victorias y Estado de cualquier Pokemón que el usuario ingrese. Permite administrar dichos Pokemones, junto con los ya cargados, para modificar alguno de los datos o eliminar un Pokemón del programa. Al iniciarse, el programa carga las 5 funciones disponibles para que el usuario decida y ejecute la que desea

## Archivos
- `listas.py`: Contiene los datos iniciales de 33 Pokemones de la Primera Generación
- `funciones.py`: Contiene las funcionalidades del programa
- `main.py`: La interfaz y menú principal


## Cómo ejecutar
El programa requiere Python. Se ejecuta desde `main.py`
